```Typing in-game chat``` **!light** `opens a small menu that will make the map look brighter by 
Using` **render.SetLightingMode**

`Activate Brightness =  0 - Default.
Bright+ = 1 - Total fullbright, similar to mat_fullbright 1 but excluding some weapon view models.
Bright++ = 2 - Increased brightness(?), models look fullbright.`
